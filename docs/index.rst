~~~~
POET
~~~~

POET (Phenotype Ontology Extract Tool) is a modern full-stack application to generate medical action annotations
for rare diseases diseases. POET provides a concise interface for scientific curators to submit MAxO annotations.

The Medical Action Ontology can be accessed at its `GitHub repository <https://github.com/monarch-initiative/MAxO>`_
and can be browsed at the `OLS Browser <https://www.ebi.ac.uk/ols4/ontologies/maxo/classes/http%253A%252F%252Fpurl.obolibrary.org%252Fobo%252FMAXO_0000001>`_.

This documentation provides background about MAxO and explains how to create MAxO annotations.
